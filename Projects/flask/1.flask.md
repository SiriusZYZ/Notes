# 1.1flask类

```python
from flask import Flask

# 当前文件
app = Flask(__name__)

# 路由
@app.route("/")
def Main_Page():
    return "<h1> This is Main page.</h1>"


@app.route("/hello")
def Page_hello():
    return "<p> This is Hello.</p>"

app.run()
```
## 1.1.1|route使用

- route 将url映射到函数中
- `@app.route("url")` 是一个装饰器，所以后接的函数会在进入url中执行
